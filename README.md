# 112 Term Project - Unfinished Duck Game
Made for the course 15-112 at CMU, spring 2020
Inspired by the Untitled Goose Game by House House
Players play as a duck trying to complete mundane tasks

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install pygame.

```bash
pip install pygame
```

## Usage
Run this file: UnfinishedDuckGame.py
